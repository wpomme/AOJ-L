# Selection Sort
- 選択ソート

```
selectionSort(A, N) // N個の要素を含む0-オリジンの配列A
  for i が 0 から N-1 まで
    minj = i
    for j が i から N-1 まで
      if A[j] < A[minj]
        minj = j
    A[i] と A[minj] を交換
```

- 設問
整列した数列を出力
２行目に整列による数値の交換回数を出力

- アルゴリズム
バブルソート・挿入ソートと同様、整列済みのデータが配列の先頭から決定されていく。
各回のループで最小の要素を持つ値を決定し、先頭にある整列ずみの要素とループで見つけた最小値を交換する。

- 特性
安定でないソート
最悪計算量 `O(N^2)`
